## Project Overview

This project was developed for **George Mason University’s IT 331: Introduction to Web Development I** course. The goal of the project was to design and build a fully functional, responsive, and interactive multi-page website while applying core front-end web development concepts and best practices.

## Features

* Responsive web design using **HTML5, CSS3, and Bootstrap**
* Interactive functionality with **JavaScript**
* DOM manipulation and event-driven features
* Dynamic content updates and animations
* Form validation and user input handling
* AJAX requests with **JSON/XML data fetching**
* REST API integration
* Consistent navigation and styling across all pages
* Accessibility and W3C validation compliance

## Technologies Used

* HTML5
* CSS3
* Bootstrap
* JavaScript
* AJAX
* JSON / XML
* REST APIs

## Learning Outcomes

Through this project, I gained hands-on experience with:

* Responsive front-end development
* Interactive web design principles
* Client-side scripting and DOM manipulation
* Working with APIs and asynchronous data
* Writing clean, organized, and maintainable code

This website was developed as an educational project for coursework purposes at George Mason University.
