SimplyRecipes
A responsive website using HTML, CSS, and JavaScript. The main page displays popular recipes which includes images of each dish, the recipe name, and the prep and cook times. Additionally, there are a set of links on the side bar that categorize each recipe by a food group. Clicking on any of the recipe cards will lead to the same recipe instructions. Other important tabs can be found on the navigation bar of each page including Contact and About.

Note: Clicking the Home tab or the SimplyRecipes Logo will take the user to the main page.
